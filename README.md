# Todo Microservices Project

Microservices based Todo Application.

## Services

### Auth Service
- Registration
- Login
- BCrypt Password Encryption
- JWT Generation

### Todo Service
- Create Todo
- View Todos
- Delete Todos
- JWT Validation

### API Gateway
- Routing
- CORS Configuration

### React Frontend
- React
- TypeScript
- Redux Toolkit
- React Router
- Tailwind CSS

## Tech Stack

Backend:
- Java 17
- Spring Boot
- PostgreSQL

Frontend:
- React
- TypeScript
- Redux Toolkit

Infrastructure:
- Docker (coming next)
- Kubernetes (AKS deployment)
